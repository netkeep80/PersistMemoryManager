Detailed Description

template<typename AddressTraitsT>
struct pmm::LinkedListNode< AddressTraitsT >

Узел двухсвязного списка для адресного пространства ПАП.

Template Parameters
AddressTraitsTTraits адресного пространства (из address_traits.h). Определяет тип индексных полей prev_offset / next_offset.

Поля хранят гранульные индексы блоков-соседей в ПАП. Sentinel «нет соседа» = AddressTraitsT::no_block.
