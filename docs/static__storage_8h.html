Detailed Description

StaticStorage — статический бэкенд хранилища ПАП (Issue #87 Phase 5).

Хранит управляемую область в фиксированном буфере размером Size байт, расположенном внутри объекта (подходит для глобальных объектов и стека).

Ограничения:

  • Размер фиксирован на этапе компиляции — expand() всегда возвращает false.
  • Не владеет памятью в смысле dynamic allocation — owns_memory() == false.
  • Требует выравнивания буфера по AddressTraitsT::granule_size.

Применение: embedded-системы, тесты с маленьким фиксированным пулом, быстрые unit-тесты без динамической памяти.
