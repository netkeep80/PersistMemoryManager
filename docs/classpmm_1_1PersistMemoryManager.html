Detailed Description

template<typename ConfigT = CacheManagerConfig, std::size_t InstanceId = 0>
class pmm::PersistMemoryManager< ConfigT, InstanceId >

Унифицированный статический менеджер персистентной памяти (Issue #110).

Все состояние и методы статические — нет необходимости создавать экземпляры. Параметр InstanceId обеспечивает уникальность типа для каждого логического «экземпляра» при одинаковой конфигурации, что реализует паттерн мультитон.

Template Parameters
ConfigTКонфигурация менеджера (например pmm::CacheManagerConfig). Должна предоставлять:
  • address_traits — тип адресных traits
  • storage_backend — тип бэкенда хранилища
  • free_block_tree — тип политики дерева свободных блоков
  • lock_policy — политика блокировок
InstanceIdИдентификатор экземпляра (по умолчанию 0). Позволяет создать несколько независимых менеджеров одной конфигурации.
Note
Состояние хранится в статических переменных класса. Каждая специализация PersistMemoryManager<ConfigT, InstanceId> имеет собственный независимый статический бэкенд, мьютекс и флаг инициализации.
Используйте destroy() перед повторной инициализацией и между тестами.
