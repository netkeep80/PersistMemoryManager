Detailed Description

Shared AVL tree helper functions for pmap and pstringview (Issue #155).

Provides a set of free static template functions implementing the core AVL tree operations (height, balance factor, rotations, rebalancing) that are shared between pmap<_K,_V,ManagerT> and pstringview<ManagerT>.

Both pmap and pstringview use the same AVL logic via pptr<T,ManagerT> — the only difference is the pptr type and the comparison key. This header factors out the common tree structure code, eliminating ~130 lines of duplication.

Template parameter PPtr must support:

  • is_null()
  • get_tree_left(), get_tree_right(), get_tree_parent()
  • set_tree_left(), set_tree_right(), set_tree_parent()
  • get_tree_height(), set_tree_height()
  • offset()
