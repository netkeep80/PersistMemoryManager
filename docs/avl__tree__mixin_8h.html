Detailed Description

Shared AVL tree helper functions for pmap and pstringview (Issue #155, #162).

Provides a set of free static template functions implementing the core AVL tree operations (height, balance factor, rotations, rebalancing, insert, find) that are shared between pmap<_K,_V,ManagerT> and pstringview<ManagerT>.

Both pmap and pstringview use the same AVL logic via pptr<T,ManagerT> — the only difference is the pptr type and the comparison key. This header factors out the common tree structure code, eliminating ~130 lines of duplication.

Template parameter PPtr must support:

  • is_null()
  • tree_node() returning a TreeNode& with get_left/set_left/get_right/set_right/ get_parent/set_parent/get_height/set_height
  • offset()
  • PPtr::manager_type::address_traits::no_block sentinel
