Detailed Description

template<typename T, typename ManagerT>
class pmm::typed_guard< T, ManagerT >

RAII scope-guard for persistent typed objects.

Calls the appropriate cleanup method (free_data() or free_all()) and then ManagerT::destroy_typed() when the guard goes out of scope.

Template Parameters
TThe persistent container type (e.g., pstring, parray, ppool).
ManagerTThe PersistMemoryManager type.
