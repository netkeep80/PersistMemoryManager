Detailed Description

template<typename T, typename ManagerT>
struct pmm::pvector< T, ManagerT >::iterator

Простой итератор для обхода вектора в порядке индексов (in-order).

Реализует in-order обход AVL-дерева (левый → корень → правый), что соответствует порядку элементов вектора.
