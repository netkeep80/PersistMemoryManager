Detailed Description

template<typename AddressTraitsT, std::size_t BufferSize, std::size_t GrowNum = 3, std::size_t GrowDen = 2>
struct pmm::StaticConfig< AddressTraitsT, BufferSize, GrowNum, GrowDen >

Базовый шаблон конфигурации менеджера со StaticStorage.

Устраняет дублирование между SmallEmbeddedStaticConfig и EmbeddedStaticConfig. Аналогичен BasicConfig, но использует StaticStorage вместо HeapStorage.

Template Parameters
AddressTraitsTТип адресного пространства.
BufferSizeРазмер статического буфера в байтах (кратно granule_size).
GrowNumЧислитель коэффициента роста (по умолчанию 3).
GrowDenЗнаменатель коэффициента роста (по умолчанию 2).
