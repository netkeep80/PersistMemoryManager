Detailed Description

template<typename AddressTraitsT = DefaultAddressTraits>
class pmm::MMapStorage< AddressTraitsT >

Бэкенд хранилища через отображение файла в память.

Открывает файл, отображает его в адресное пространство процесса. При создании нового файла инициализирует его нулями.
