Detailed Description

template<typename AddressTraitsT>
class pmm::BlockStateBase< AddressTraitsT >

Базовый класс блока для state machine.

Наследует Block (LinkedListNode + TreeNode). Все поля приватные для потомков. Доступ только через методы состояний.

Layout Block (32 bytes при DefaultAddressTraits): [0..7] LinkedListNode: prev_offset (4), next_offset (4) [8..31] TreeNode: weight (4), left_offset (4), right_offset (4), parent_offset (4), root_offset (4), avl_height (2), node_type (2)
