struct  ManagerHeader
 Manager header parameterized on AddressTraitsT (Issue #175: 64-bit index support). All _offset and counter fields use index_type, so LargeDBConfig uses uint64_t indices and DefaultAddressTraits keeps uint32_t with exactly 64 bytes as before. prev_total_size is runtime-only (zeroed by load() — not persisted).
