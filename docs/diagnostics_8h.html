Detailed Description

Structured diagnostics for verify and repair modes (Issue #245).

Provides:

  • RecoveryMode — enum to select verify-only vs. repair behaviour
  • DiagnosticEntry — single violation record with type, affected block, action
  • VerifyResult — aggregated result of a verify or repair pass

The verify mode performs read-only diagnostics without modifying the image. The repair mode performs the same diagnostics, then applies documented fixes. Both modes populate a VerifyResult with structured diagnostic entries.
