Detailed Description

Политика логирования в stderr.

Выводит ошибки и ключевые события менеджера в стандартный поток ошибок. Полезна для отладки и диагностики.

using MyConfig = pmm::BasicConfig<
pmm::DefaultAddressTraits,
pmm::config::NoLock,
5, 4, 64,
pmm::logging::StderrLogging // включить логирование в stderr
>;
using MyMgr = pmm::PersistMemoryManager<MyConfig>;
