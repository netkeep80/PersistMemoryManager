Detailed Description

template<typename T, typename ManagerT>
struct pmm::parray< T, ManagerT >

Persistent dynamic array with O(1) random access (Issue #195, Phase 3.2).

Stores a header (size, capacity, data block index) in PAP. Element data is stored in a separate contiguous block allocated via the manager.

A parray object is created in PAP via create_typed<parray<T>>() and destroyed via destroy_typed() after calling free_data().

Invariants:

  • If _data_idx != 0, the data block contains _size elements of type T.
  • _capacity >= _size (there is room for at least _size elements).
  • When _data_idx == 0, the array is empty.
  • Element type T must be trivially copyable.

Layout in PAP:

parray<T> (in PAP) Data block (in PAP)
+----------------+ +--------------------------+
| _size: u32 | | T[0] | T[1] | ... | T[n]|
| _capacity: u32 |---idx---> | |
| _data_idx | +--------------------------+
+----------------+
