Detailed Description

PMM single-header preset: SingleThreadedHeap — NoLock + HeapStorage, grow 25% (CacheManagerConfig)

Self-contained single-header file generated from the PersistMemoryManager library (https://github.com/netkeep80/PersistMemoryManager).

Drop this file into your project and include it — no other files needed.

Usage:

#include "pmm_single_threaded_heap.h"
pmm::presets::SingleThreadedHeap::create(64 * 1024);
void* ptr = pmm::presets::SingleThreadedHeap::allocate(256);
pmm::presets::SingleThreadedHeap::deallocate(ptr);
// Persistence (save/load):
pmm::save_manager<pmm::presets::SingleThreadedHeap>("heap.dat");
pmm::load_manager_from_file<pmm::presets::SingleThreadedHeap>("heap.dat");
