◆ weight

template<typename AddressTraitsT >
index_type pmm::TreeNode< AddressTraitsT >::weight
protected

Universal granule-key / granule-scalar. First field for cache-efficient access (Issue #126). Semantics determined by the owning forest domain:

  • free-tree: state discriminator (0 = free); sort key derived from PAP geometry.
  • user domains: domain-specific sort key (e.g., granule count, string index).
