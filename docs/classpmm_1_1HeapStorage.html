inlinenoexcept

Расширить буфер на additional_bytes байт (минимум).

Выделяет новый буфер с ростом >= additional_bytes, копирует данные. Старый буфер освобождается (если owns_memory == true).

Parameters
additional_bytesМинимальный прирост в байтах.
Returns
true при успехе, false при ошибке.
