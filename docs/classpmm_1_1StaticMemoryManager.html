Detailed Description

template<typename ConfigT = CacheManagerConfig, std::size_t InstanceId = 0>
class pmm::StaticMemoryManager< ConfigT, InstanceId >

Статический менеджер персистентной памяти (Issue #108).

Все состояние и методы статические — нет необходимости создавать экземпляры. Параметр InstanceId обеспечивает уникальность типа для каждого логического «экземпляра» при одинаковой конфигурации, что реализует паттерн мультитон.

Template Parameters
ConfigTКонфигурация менеджера (например pmm::CacheManagerConfig). Должна предоставлять lock_policy.
InstanceIdИдентификатор экземпляра (по умолчанию 0). Позволяет создать несколько независимых менеджеров одной конфигурации.
Note
Состояние хранится в статических переменных класса. Каждая специализация StaticMemoryManager<ConfigT, InstanceId> имеет собственный независимый статический бэкенд, мьютекс и флаг инициализации.
Используйте destroy() перед повторной инициализацией и между тестами.
