Detailed Description

template<typename IndexT, std::size_t GranuleSz>
struct pmm::AddressTraits< IndexT, GranuleSz >

Traits адресного пространства ПАП.

Template Parameters
IndexTТип гранульного индекса (uint8_t, uint16_t, uint32_t, uint64_t).
GranuleSzРазмер гранулы в байтах (степень двойки ≥ 1).

Статические проверки (static_assert):

  • IndexT — беззнаковый целый тип.
  • GranuleSz — степень двойки (GranuleSz & (GranuleSz-1) == 0).
  • GranuleSz ≥ 1.
