Detailed Description

template<typename _K, typename _V, typename ManagerT>
struct pmm::pmap< _K, _V, ManagerT >::iterator

Итератор для обхода словаря в порядке ключей (in-order).

Реализует in-order обход AVL-дерева (левый -> корень -> правый), что соответствует порядку возрастания ключей.
