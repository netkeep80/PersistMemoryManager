Detailed Description

Core types and constants for PersistMemoryManager.

Contains: ManagerHeader, MemoryStats, ManagerInfo, BlockView, FreeBlockView and utility functions for byte/granule conversion.

Sizes of structures are protected by static_assert: Block<DefaultAddressTraits> == 32 bytes ManagerHeader<DefaultAddressTraits> == 64 bytes
