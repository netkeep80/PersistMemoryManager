struct  BlockHeader
 Block header (32 bytes = 2 granules, Issue #59). size=0: free; size>0: data granules. root_offset=0: free-blocks tree; root_offset=own_idx: allocated block root (Issue #75). Issue #69: magic removed; validity via is_valid_block(). total_size = next_offset - idx. More...
 
struct  ManagerHeader
 Manager header (Issue #59: 64 bytes). All _offset fields are granule indices. prev_base_ptr / prev_total_size are runtime-only (nulled by load() — not persisted). More...
