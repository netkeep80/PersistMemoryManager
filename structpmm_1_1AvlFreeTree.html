Detailed Description

template<typename AddressTraitsT = DefaultAddressTraits>
struct pmm::AvlFreeTree< AddressTraitsT >

Standard AVL tree implementation for free block tree policy.

All-static class for AVL tree of free blocks (Issue #73 FR-02, AR-03). Does not depend on PersistMemoryManager singleton — takes base_ptr and header as context.

Sort key: (total_granules, block_index) — strict ordering. Best-fit search runs in O(log n).
