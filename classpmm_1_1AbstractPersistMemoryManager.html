Выделить один объект типа T и вернуть персистентный указатель pptr<T>.

Снаружи менеджера следует хранить только pptr<T>, а не сырые указатели. Для разыменования используйте resolve<T>(p) или resolve_at<T>(p, i).

Issue #97: pptr<T> — персистентный 4-байтный указатель (гранульный индекс), адресно-независимый, корректно загружается из файла по другому базовому адресу. Issue #100: возвращает Manager::pptr<T> = pmm::pptr<T, manager_type>. Неявно конвертируется в pmm::pptr<T, void> для обратной совместимости.

Template Parameters
TТип выделяемого объекта.
Returns
pptr<T> — персистентный указатель или pptr<T>() при ошибке.
