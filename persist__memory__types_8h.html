Detailed Description

Базовые типы данных и константы для PersistMemoryManager (Issue #73)

Содержит: BlockHeader, ManagerHeader, pptr<T>, MemoryStats, ManagerInfo, BlockView, FreeBlockView и вспомогательные функции конвертации.

Размеры структур защищены static_assert (Issue #59, #73 FR-03): BlockHeader == 32 bytes ManagerHeader == 64 bytes
