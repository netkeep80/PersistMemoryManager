Detailed Description

AVL-дерево свободных блоков для PersistMemoryManager (Issue #73)

Все-статический класс PersistentAvlTree для управления деревом свободных блоков. Принимает base_ptr и header* как явные параметры контекста — не имеет неявной зависимости от синглтона PersistMemoryManager (Issue #73 AR-03).

Ключ сортировки: (total_granules, block_index) — строгий порядок. Поиск best-fit выполняется за O(log n).
